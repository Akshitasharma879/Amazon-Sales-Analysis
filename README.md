Overview
This project presents a comprehensive sales analysis of Amazon products using Power BI.The dashboard provides insights into year-to-date (YTD) and quarter-to-date (QTD) sales, product performance, customer reviews, and category-wise revenue to support data-driven business decisions.

Dataset
Source: Amazon Sales Dataset
Data Type: Combined sales, product, and review data
Total Records: 89,000+ sales entries (after data cleaning & transformation)
